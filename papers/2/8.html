Säure-Base-Titration

Sie wissen um die Bedeutung von Titrationen in der Chemie

Die Titration ist eine analytische Methode zur Bestimmung der Konzentration einer unbekannten Lösung. Bei der Säure-Basen-Titration kann somit die Konzentration einer Lösung aus einer unbekannten Säure oder Base ermittelt werden

Sie können den Vorgang einer Titration erklären und beschreiben, wie die Neutralisation dafür ausgenutzt wird

Bei einer Titration wird in regelmässigen Abständen eine Masslösung aus einer starken Säure oder Base beigegeben zu einer Probelösung mit einer unbekannten Konzentration an einer Säure oder Base. Dies wird wiederholt bis sich die Lösung im Äquivalenzpunkt befindet, wo sich die Säure und Base stöchiometrisch vollständig reagiert haben. Dies kann festgestellt werden durch einen passend gewählten Farbindikator oder ein pH-Meter. Dies ist der Endpunkt der Titration.

Sie können den Äquivalenzpunkt in einer Titrationskurve bestimmen

Der Äquivalenzpunkt in einer Titrationskurve kann bestimmt werden mithilfe dessen Konvexität. Dabei ist der Äquivalenzpunkt, wo die Titrationskurve von Konvex zu Konkav umschlägt.

Sie können die Resultate einer Titration interpretieren und die Stoffmengenkonzentration einer unbekannten Säure oder Base bestimmen

Im Äquivalenzpunkt hat die ganze Probelösung mit der Masslösung reagiert. In diesem Punkt muss somit das Verhältnis der Stoffmenge der beiden Lösungen gleich dem Verhältnis ihrer beiden stöchiometrischen Koeffizienten sein in der Neutralisationsgleichung. Sei nun aaa der Koeffizient von der Säure oder Base in der Masslösung (MMM) und bbb der Koeffizient vor der Säure oder Base von der Probelösung (PPP). Dann gilt
b⋅c(M)⋅V(M)=a⋅c(P)⋅V(P) b \cdot c(M) \cdot V(M) = a \cdot c(P) \cdot V(P) b⋅c(M)⋅V(M)=a⋅c(P)⋅V(P)

Sie können Titrationskurven interpretieren

Eine Titration von einer starken Säure oder Base zeigt sich daran, dass diese nur an einem Punkt umschlagt. Dabei ist der Äquivalenzpunkt immer im Neutralpunkt also bei einem pH-Wert von 7.

Die Titrationskurve einer schwachen Säure oder Base kann erkannt werden daran, dass diese zwei Umschlagpunkte beinhaltet. Einer davon ist der Halbäquivalenzpunkt, welcher beim pKSpK_SpKS​ bzw. beim pKBpK_BpKB​ sich befindet. Der andere ist der Äquivalenzpunkt, welcher sich nicht beim Neutralpunkt befindet.

Pufferlösungen

Sie können die Wirkungsweise eines Puffers erklären

Eine wässrige Lösung, welche den pH-Wert trotz Störung auf einem bestimmten Wert konstant haltet, bezeichnet man als Pufferlösung. Eine Pufferlösung kann gebildet werden, indem man einer Lösung ein Säure-Base-Paar hinzufügt. Dabei muss dies eine schwache Säure (HPu\text{HPu}HPu) und schwache Base (Pu−\text{Pu}^-Pu−) sein, welche ebenfalls stärker als Wasser sein müssen. Dies ergibt somit die folgenden linksliegenden Gleichgewichte.
HPu+H2O⇌Pu−+H3O+ \text{HPu} + \text{H}_2\text{O} \rightleftharpoons \text{Pu}^- + \text{H}_3\text{O}^+ HPu+H2​O⇌Pu−+H3​O+

Pu−+H2O⇌HPu+OH− \text{Pu}^- + \text{H}_2\text{O} \rightleftharpoons \text{HPu} + \text{OH}^- Pu−+H2​O⇌HPu+OH−

Im Fall der Zugabe von einer starken Säure bzw. Base reagiert diese dann nicht mit dem Wasser, sondern mit der schwachen Base bzw. Säure. Dies verändert den pH-Wert nicht direkt, da kein H3O+\text{H}_3\text{O}^+H3​O+ und OH−\text{OH}^-OH− produziert wird. Die Menge an der schwachen Säure oder Base verändert sich jedoch, weshalb die oben aufgeführten Gleichgewichte leicht nach rechts rutschen.

Sie können den pH-Wert von gepufferten Lösungen mit der Hendersson-Hasselbalch-Gleichung berechnen

Sei einer Lösung ein Säure-Base-Paar aus einer schwachen Säure (HPu\text{HPu}HPu) und schwache Base (Pu−\text{Pu}^-Pu−) hinzugefügt worden sein. Dann gilt
