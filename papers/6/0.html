Sets

You know what a set is

A set is collection of distinct objects. These objects are called the elements or members of the set. A set is defined as a list of these elements encapsulated with these symbols {yourelemts}\{ your elemts \}{yourelemts} and every element is separated with a , so the set of the numbers 3 and 4 is written in the roster notation as follows {3,4}\{3, 4\}{3,4}.

If you have a big set that follows a clear pattern, you can use ... to symbolize that the set continuous. Consider for example a set with all the positive integers. In words, you would say that this set contains the numbers 1, 2, 3 and so on. To convert this into a mathematic expression, you just have to substitute and so on for .... Following this, you would end up with a set that looks like this {1,2,3,...}\{1, 2, 3, ...\}{1,2,3,...}. You can also do this in the other direction, like for example {...,−2,−1,0,1,2,...}\{..., -2, -1, 0, 1, 2, ...\}{...,−2,−1,0,1,2,...} this set would represent every integer.

There also exists an empty set that can be written as {}\{\}{} or ∅\emptyset∅.

You know the operations that you can perform on sets

Every of the following examples has two sets A={2,4,6,10}A = \{2, 4, 6, 10\}A={2,4,6,10} and B={2,5,10,20}B = \{2, 5, 10, 20\}B={2,5,10,20}

  • An union between two sets is all of their elements combined into one set and can be written like this A∪BA \cup BA∪B, pronounced as A-cup-B. With the example definitions above, this would be equal to {2,4,5,6,10,20}\{2, 4, 5, 6, 10, 20\}{2,4,5,6,10,20}.

  • Computing the intersection of two sets results in a set containing elements that both sets have in common and can be written like this A∩BA \cap BA∩B, pronounced as A-cap-B. With the example definitions above, this would equal {2,10}\{2, 10\}{2,10}.

  • The difference between two sets results in one set that contains every element of the first set that is not in the second one. This can be written as A∖BA \setminus BA∖B, pronounced as A-minus-B. With the example definition above, this would equal {4,6}\{4, 6\}{4,6}.

Positional notation of numbers

Binary into Decimal

You can convert Binary numbers into decimal number with these simple steps

  • Only to do at the beginning: Take the first binary digit

  • Double your number and add the next binary digit

  • Continue like this

This can be shown in a table going from left to right. In the first row we write all the digits of the binary number and in the bottom row we will keep track of the current result of our computations.

binary 1 1 0 1
decimal 1 3 6 13

The bottom right cell is now your result. If you don’t completely understand the procedure, this is what is calculated

  • Take the first digit, in this case 111

  • Double it and add the next digit, in this case also 111 and write down the temporary result 333 right from the previous temporary result

  • Take 333 double it and add the next digit and your result in 666

  • Then double it again and add the last digit, leaving you with 131313

Decimal into Binary

This can be solved very similarly to the last problem by just going backward.

  • Only to do at the beginning: Take your decimal number and write it in the bottom right cell

  • Then divide it by two and write the rounded solution left from the previous one. Always round down, so 6.56.56.5 for example, becomes 666. Write the remainder of the division left from your last binary number. For example, the reminder of 7/27 / 27/2 is 111 because 3∗2+1=73 * 2 + 1 = 73∗2+1=7.

  • Do this until your rounded quotient becomes 000. Take the division 1/21/21/2 for example, it has the result of it 000 with the reminder of 111.

The same example as in the table above can be explained with these steps

  • Take 131313 and write it down and divide it by 222 this will equal 6.56.56.5. So the remainder of this division is 111 so you write down an 1 for the first binary digit. 6.56.56.5 rounded down is 666, so you write that down left from 131313.

  • Divide 666 by 222 this equals 333 and the remainder is 000 So you write down 000 in the binary row and a 333 left from the 666.

  • Divide 333 by 222 and you get 1.51.51.5 and a remainder of 111 so you can write down a 000 in the binary row and a 111 in the decimal row.

  • Divide 111 by 222 and you get 000 with the remainder of 111. So you just have to write down the 111 in the binary row and because the result was 000 you can stop now.

Where aaa is any number equal or bigger than 111 but smaller than 101010. bbb can be any negative or positive integer. If you want to write the number 0.0000000052340.0000000052340.000000005234 for example in scientific notation. You have to move the decimal point until the new number satisfies the condition for aaa. In this case you have to move it 999 times to the right resulting in 5.2345.2345.234, this number becomes your aaa. When you put this together, you should be left with

The exponent (bbb) has a minus in this example because we moved the decimal point to the right. When we would want to write a very big number in scientific notation, then we would move the decimal point to the left and bbb would become positive.

Basic arithmetic with some help of geometry

Addition and subtraction

Let’s imagine a point on the number line that represents the number 333 now if we want to add the number +2+2+2 to it, we draw an arrow from 333 pointing to the right with the length of 222. This addition can be representing as shifting the entire number line from the origin of the arrow to its end. Please note that we are just shifting the line with our number 333 on it, our scale stays where it is. So if we just shift the line by 222 then our original number 333 ends up on 555.

Subtracting would be the same operation, just with our arrow pointing left.

Multiplication

Let’s imagine the number 444 as a point on the number line, and we want to multiply it with the number 222. To achieve this we can stretch the number line by a factor of 222 in both direction from the origin with our point representing the number 444 on it. But same as with the addition, the scale stays in place. So when we stretch the number line with a factor of 222 then the point that before represented 444 now ends up on 888.

Multiplication with a negative number can be simplified into two operation. First we have to flip the entire number line around it origin, this operation accounts for the −-− sign. Then we can stretch it as described above. Following this, multiplying a number with −1-1−1 and then again with −1-1−1 will give us a positive number because we flip the number line twice.

Where xxx and yyy are variables. The factor aaa is the slope and bbb is the Y-intercept.

The slope of a linear equation defines by how much does the y-values changes between two points that are 111 unit away from each other on the x-axis.

Let’s define two points P=(4,4)P = (4, 4)P=(4,4) and Q=(8,2)Q = (8, 2)Q=(8,2)
